Title: 
Task Tracker

How to run:
After downloading the code, set up a local server in VS to avoid issues with fetch(). To set up a local server, type "-m http.server" in the terminal in VS. Then, type http://localhost:8000/ in your browser.

View inventory:
The new task and edit categories buttons at the top expand a modal to allow the user to edit and add items. The completed tasks button at the bottom allows the user to see a log of completed tasks.

Known issues or limitations:
The buttons at the bottom of each card get cut off. The items inside are not structured properly. Also, no data connects with each other. Also, currently the only way to access the login screen is by opening the file.

Changes from Phase 1 wireframes:
I am going with a card view instead of a table view, mainly to make it look better on mobile. I am also using modals instead of different pages to keep everything on one page for simplicity.
